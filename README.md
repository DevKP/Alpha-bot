# Alpha-bot

## Install

### 
```bash
pip install virtualenvwrapper
```

```bash
export WORKON_HOME=$HOME/.virtualenvs
source /usr/local/bin/virtualenvwrapper.sh
```

```bash
source ~/.bashrc
mkvirtualenv -p /path/to/python3 bot
```

```bash
(bot) pip install -r requirements.txt
```

## Run tests
```bash
(bot) pytest tests.py
```
